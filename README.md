BLT-APP

Laravel 10 dengan MYSQL Database

Import Database ke MYSQL terlebih dahulu sebelum menjalankan aplikasi ini.
Lakukan **composer install** untuk menginstall semua dependency yang dibutuhkan.
Lalukan **PHP Artisan migrate**.

jalankan perintah berikut di terminal:
**Php artisan serve** untuk menjalankan aplikasi

Untuk mengganti hak akses harus ketik langsung di route /hak-akses dengan syarat harus sudah login.
